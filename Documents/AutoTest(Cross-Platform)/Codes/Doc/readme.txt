数据库和表要自己建好

数据库名称：testdb
表名：result

字段：	名称		类型
	moudle		varchar(128)
	name		varchar(128)
	date		date
	url		    varchar(256)
	response	mediumtext

建表sql：
CREATE TABLE `result` (  `moudle` varchar(128) DEFAULT NULL,  `name` varchar(128) DEFAULT NULL,  
`date` varchar(128) DEFAULT NULL,  `url` varchar(256) DEFAULT NULL,  `response` mediumtext
) ENGINE=InnoDB DEFAULT CHARSET=utf8



依赖包：
python-intervals
requests
configparser
pymysql
xlrd
xlwt

附带依赖包使用方法：
把依赖包解压复制到/usr/local/python36/lib/python3.6/site-packages/下就可以了，其中/usr/local/python36/是安装路径
